# PureVPN 风格界面设计分析

## 参考截图关键元素

### 整体风格
- 深色主题（深紫色/深蓝色背景）
- 科技感强，简洁现代
- 渐变色彩效果

### 客户端界面布局

#### 左侧菜单栏
- Dashboard（仪表盘）
- Locations（位置/节点）
- Remove My Data
- Password Manager
- Dark Web Monitoring
- Discover More
- Refer a Friend
- Support
- Profile
- Preferences

#### 主界面区域
- 顶部：当前连接节点信息（国旗 + 城市名，如 Frankfurt, DE）
- VPN IP 显示
- 连接时长计时器（03:23:56）
- 大型圆形连接按钮（紫色渐变，带电源图标）
- "Connected" 状态文字
- 底部流量统计：
  - Trackers & Ads Blocked: 17
  - Uploaded: 17.67 MB
  - Downloaded: 2.3 MB
- Security Progress 进度条
- Connection Details 展开按钮

#### 右侧面板
- 快速节点选择（Singapore 等）
- 节点列表

### 配色方案
- 主背景：#1a1a2e 或类似深色
- 强调色：紫色渐变 #6c5ce7 → #a29bfe
- 连接按钮：紫色圆形
- 文字：白色/浅灰色
- 图标：简洁线条风格

### Web 前端需求
- 首页：产品介绍 + 下载按钮
- 登录/注册页面
- 用户控制台
- 充值页面（收款码占位）
- 客户端下载页面

### 后台管理需求
- 用户列表
- 手动激活/延长订阅按钮
- 订单管理
- 系统设置
