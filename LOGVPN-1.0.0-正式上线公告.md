## 📊 系统架构

```
用户设备
    │
    ├─ Web浏览器 → http://155.94.160.248
    ├─ 桌面客户端 → Clash核心 → 节点
    └─ 移动App → 系统VPN → 节点

服务器架构
    │
    ├─ Nginx (80端口)
    │   ├─ / → 前端静态文件
    │   └─ /api/ → Node.js后端(3000端口)
    │
    ├─ Node.js API
    │   ├─ JWT认证
    │   ├─ 节点管理
    │   ├─ 流量统计
    │   └─ 订阅生成
    │
    └─ SQLite数据库
        ├─ users
        ├─ subscriptions
        ├─ nodes
        └─ traffic_logs
```
